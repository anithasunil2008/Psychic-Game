About The Game:


The Psychic Game A simple game that allows you to guess the letter the computer chooses.

Choose a letter from A - Z and test your psycic ability to see if you can read a computers mind!!!

Instructions:

  • Try to guess the letter the computer has picked. Any letter from A - Z
  • You can guess 10 times
  • To start the game click the start button
